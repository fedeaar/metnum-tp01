## Page Rank

### Crear el ejecutable para la experimentacion:

desde la raiz del repo:

// nos movemos a ./experimentos
> cd ./experimentos
 
// creamos la carpeta del ejecutable
> mkdir build
    
// entramos
> cd build

// abrimos bash (si en windows)
> bash

// creamos el cmake
$ cmake ../../implementacion

// creamos el ejecutable	
$ make ./pagerank
